# Moive-Recommendation-Engine

Movie recommendation engine: **FastAPI** API, **React 18** (Vite) UI, **Surprise SVD** model, **PostgreSQL** (Docker or Neon), **Alembic** migrations.

## Quick start

1. Copy `.env.example` to `backend/.env` and adjust if needed.
2. From the repo root: `docker compose up --build`
3. Open **http://localhost:5173** (API docs: **http://localhost:8000/docs**).
